## Installation

Using [lazy.nvim](https://github.com/folke/lazy.nvim):

```lua
{
  "ubaldobearki/tonada.nvim",
  config = function()
    vim.cmd("colorscheme tonada")
  end
},
```

* [Treesitter](https://github.com/nvim-treesitter/nvim-treesitter)
